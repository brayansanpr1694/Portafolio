My Favorite Tools

Published June 3, 2025 · By Brayan Santamaria

Over the years, I’ve tried many tools, editors, and frameworks. But some have stuck with me and become essential to my daily workflow. Here are my favorites — the ones I reach for every day.

VSCode: My Home Base

Visual Studio Code is my go-to code editor. With its clean interface, rich extension ecosystem, and blazing fast performance, it's hard to beat. I use extensions for Dart, Python, GitLens, Prettier, and more.

"If I could only install one app on a new machine, it would be VSCode — no question."

Git: Version Control Superpowers

Whether I’m solo or collaborating, Git is non-negotiable. Branching, merging, version history — it’s all essential. I mostly use Git from the terminal, but occasionally rely on tools like GitKraken or the VSCode GUI for visual clarity.

Dart & Flutter: Build Once, Run Everywhere

Ever since I discovered Flutter, Dart has become part of my daily language set. I love how expressive it is, and Flutter’s widget system makes building beautiful UIs feel like stacking LEGO blocks.

Python: My Utility Belt

When it comes to quick scripts, data manipulation, or automation — Python is my hero. It’s readable, flexible, and has a package for almost anything I need. Whether it's a CLI tool or backend logic, Python delivers.

Terminal: Power at My Fingertips

I’m a big fan of terminal tools. I use Zsh with Oh My Zsh, and I’ve customized it with plugins for Git and syntax highlighting. It keeps my workflow snappy and efficient.

Bonus Tools

  • Postman – for API testing and debugging
  • Figma – to collaborate with designers or quickly mock up UI ideas
  • Notion – to manage tasks, notes, and documentation
  • Docker – when I need to containerize and ship apps consistently

Conclusion

Great tools don’t write code for you, but they do make the journey smoother and more enjoyable. These tools help me stay focused, organized, and creative — and I’m always on the lookout for new ones that might become part of the toolkit.
